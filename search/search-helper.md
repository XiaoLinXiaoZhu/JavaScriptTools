# 搜索引擎快捷跳转

在百度、Google、搜狗、360、Bing 等搜索引擎页面右侧显示浮动按钮，一键将当前搜索词切换到其他搜索引擎。

## 功能

- 自动检测当前搜索词
- 显示其他搜索引擎的快捷按钮
- 点击按钮在新标签页打开对应搜索引擎的搜索结果
- 半透明设计，鼠标悬停高亮，不干扰浏览

## 支持的搜索引擎

| 引擎 | 域名 |
|------|------|
| 百度 | www.baidu.com |
| Google | www.google.com |
| 搜狗 | www.sogou.com |
| 360 | www.so.com |
| Bing | cn.bing.com / www.bing.com |
