# Danbooru Tag Copy

在 Danbooru 帖子页面一键复制所有标签，按分类格式化为五行输出。

## 输出格式

```
meta tag1, meta tag2, ...
rating,
character tag1, copyright tag1, ...
@artist1, @artist2, ...
general tag1, general tag2, ...
```

- 逗号分隔，行末带逗号
- `_` 替换为空格，全部小写
- 不存在的分类留空行
