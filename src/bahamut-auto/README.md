# 动画疯自动脚本

动画疯自动同意年龄确认、移除广告跳转、广告静音、自动点击跳过广告。

借鉴了 [luosansui 的脚本](https://greasyfork.org/zh-CN/scripts/484045) 的部分逻辑，但原脚本部分功能已失效，且切换剧集时不会重新加载。本脚本加入了监控机制，能够检测剧集变化并重新执行，避免频繁刷新。

可以按 F12 查看控制台 debug 信息，跟踪运行状态。
