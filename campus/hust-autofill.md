# HUST 军理线上作业自动填充助手

为 HUSTer 编写的小工具，可以一键填写华中出版社的习题（仅限单选题和多选题）。

脚本中内置了军理答案，不能确保所有人都适用，可以按照实际情况修改录入的答案。

> 注意：录入答案时，只需按照评分结果页面的题目顺序填入 answer 即可。网页会将部分题目进行打乱，脚本中已包含归正功能。

## 使用方法

1. 安装油猴脚本
2. 打开军理 PC 端页面：[华中科技大学出版社 - 大学军事理论教程](http://bookcenter.hustp.com/detail/513.html)
3. 选择试卷进行答题
4. 点击右下角的"点我自动填充答案"即可

另有智慧校园自动翻页功能（[GreasyFork](https://greasyfork.org/scripts/497897)）。
