# ASMR 自动下载工具

用于在使用 Alist 驱动的网站下自动下载单个 zip 压缩包和分 part 的 rar 压缩包。

打开对应网址即可开始自动下载，目前支持的网站：

- [asmrgay](https://www.asmrgay.com/)

请合理使用，适量放慢多次下载的间距，避免被拦截。

## 安装

[从 GreasyFork 安装](https://greasyfork.org/scripts/484096) | [从 GitHub Pages 安装](https://xiaolinxiaozhu.github.io/JavaScriptTools/tools/asmr-download.user.js)
